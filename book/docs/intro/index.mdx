# Welcome to Physical AI & Humanoid Robotics

Welcome to your comprehensive guide for building intelligent, embodied robotic systems.

## What You'll Learn

This book takes you on a hands-on journey through the complete stack of physical AI and humanoid robotics:

- **Module 1: ROS 2 Foundations** - Master the Robot Operating System 2, the industry-standard framework for robotics development
- **Module 2: Simulation & Digital Twins** - Build and test robots in realistic virtual environments before deploying to hardware
- **Module 3: NVIDIA Isaac Platform** - Leverage GPU-accelerated perception, planning, and AI for robotics
- **Module 4: Humanoid Robotics** - Design and program bipedal robots with human-like capabilities

## Who This Book Is For

This book is designed for:

- Software engineers transitioning into robotics
- Robotics students seeking hands-on experience
- AI/ML engineers interested in embodied intelligence
- Hobbyists ready to build sophisticated robotic systems

## Prerequisites

To get the most out of this book, you should have:

- Basic Python programming experience
- Familiarity with Linux command line
- Understanding of fundamental programming concepts
- Enthusiasm for building robots!

## How to Use This Book

Each chapter builds on previous concepts, so we recommend following the modules in order. However, experienced practitioners can jump to specific topics of interest.
